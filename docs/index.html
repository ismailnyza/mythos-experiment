Smart File Organizer CLI

Automatically organize files by type, date, or custom rules. Saves hours of manual work.

Why This Tool?

This command-line tool helps developers, sysadmins, and anyone dealing with messy directories. It automatically sorts files into logical folders, renames duplicates, and keeps your workspace clean.

  • Free and open source - core version available on GitHub
  • Cross-platform - works on macOS, Linux, Windows
  • Zero dependencies - just Node.js runtime
  • Customizable rules - adapt to your workflow
