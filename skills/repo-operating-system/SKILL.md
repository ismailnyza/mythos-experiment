# Repo Operating System

Use this skill whenever you start or resume work in this workspace.

## Steps
1. Read AGENTS.md, GOAL.md, MISSION_STATUS.md, TASK_BOARD.md, PROJECT_CONTEXT.md, PERMISSIONS.md, TOOLS.md
2. Rewrite the current objective in your own words
3. Convert the goal into a task list
4. Execute the top task
5. Update MISSION_STATUS.md and TASK_BOARD.md
6. Repeat until success criteria are satisfied or a true blocker exists

## Rules
- Never say the mission is complete without verification
- Keep status files fresh
- Make progress, do not hover
